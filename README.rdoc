== Приложение для brinchurch.pro

brinchurch.pro - это моя персанальная страница. Это простое приложение является моим
первым проектом на Ruby on Rails. Я не использовал какие либо готовые CMS, потмоу что
хочу научиться использованию Ruby on Rails на примере действительно полезного проекта.
Этот проект для меня таковым и является.
